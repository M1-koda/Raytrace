### 説明
本プログラムは、レイトレーシングと呼ばれる方法を用いて指定された物体(rt_file)のレンダリングを行っている。</br>
レイトレーシングは光線による追跡を行い、物体による光の反射を計算することでカメラの位置から見える物体の3D表現をしみゅれーｔ
